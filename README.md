# 猫の飼育準備診断サイト

## 1. 概要

**【どんなサイトか】**
猫を迎える前に生じる「費用・時間・生活制約」などの現実的なハードルを可視化し、飼育の準備度を測る診断サービスです。

<img width="1156" height="706" alt="git_img" src="https://github.com/user-attachments/assets/2e727638-0e14-4439-82d2-ae8692171de3" />

**【何ができるのか】**
6つの具体的な質問に答えることで準備度をスコアリングし、全ユーザーのデータを基にした相対評価（上位◯%など）と客観的なフィードバックを提示します。

**【アピールポイント】**
「とりあえず試したい」ユーザーのためにゲスト診断を可能にしつつ、その後の会員登録時に診断データを自動で引き継ぐフローを実装するなど、ユーザーの離脱を防ぐUI/UXとロジック設計にこだわっています。

## 2. 使用技術
- **バックエンド:** PHP 8.2 / Laravel 12.0
- **フロントエンド:** HTML / CSS / Tailwind CSS / Livewire 3.x / Alpine.js
- **UIコンポーネント:** Flux UI
- **データベース:** PostgreSQL 17
- **開発環境:** Docker Desktop / Laravel Sail
- **その他ツール:** Git / GitHub

## 3. 機能・非機能一覧

### 【機能一覧】
*ライブラリ（パッケージ）を活用し、セキュアかつスムーズな操作性を実現しています。*

- **ユーザー認証機能**（Laravel Breeze）
  - 新規登録、ログイン、ログアウト
- **準備度診断機能**（Livewire）
  - ページ遷移なし（非同期）でのスムーズなフォーム回答
  - 独自の重み付けロジックによるスコア計算
- **相対評価・フィードバック表示機能**
  - 全ユーザーの平均点やパーセンタイル（上位◯%）の動的集計
- **ゲストデータの引き継ぎ機能**
  - セッションを活用し、未ログイン時の診断結果を新規登録後に自動紐付け
- **診断履歴の管理機能**
  - 過去の診断結果の一覧表示、前回スコアとの差分比較
